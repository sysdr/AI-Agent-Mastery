# Secure Memory & Context Systems

Production-grade AI Agent memory system with encryption, PII detection, and context optimization.

## Features
- Encrypted SQLite conversation storage
- Real-time PII detection and classification
- Context window optimization with cost management
- Comprehensive audit logging
- Security monitoring dashboard

## Quick Start
```bash
./scripts/start.sh
```

## Tech Stack
- Backend: Python 3.12, FastAPI, SQLCipher, Gemini AI
- Frontend: React 18, TypeScript, Tailwind CSS
- Database: SQLite with SQLCipher encryption
